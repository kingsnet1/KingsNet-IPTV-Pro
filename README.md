# KingsNet-IPTV-Pro
KingsNet IPTV Pro is a modern, feature-rich web application for streaming IPTV playlists with an intuitive channel-based interface. Built with pure HTML, CSS, and JavaScript, this progressive web app delivers a seamless IPTV viewing experience directly in your browser.

 Features

路  Smart Channel Grid - Visual thumbnails and organized categories for easy browsing
路  Advanced Video Player - Full-featured player with fullscreen toggle and controls
路  M3U Playlist Support - Load playlists from URLs or local files
路  Category Filtering - Organize channels by categories (Sports, News, Entertainment, etc.)
路  Responsive Design - Works perfectly on desktop, tablet, and mobile devices
路  One-Click Playback - Instant channel switching with visual feedback
路  Live Search & Filter - Quickly find your favorite channels
路  Local Storage - Remembers your preferences and recent playlists

 Technology Stack

路 Frontend: Pure HTML5, CSS3, JavaScript (ES6+)
路 Video: HTML5 Video Element with HLS.js support
路 Styling: CSS Grid & Flexbox with modern dark theme
路 Storage: LocalStorage for user preferences
路 Compatibility: All modern browsers with CORS support

 Quick Start

1. Clone the repository
  
   git clone https://github.com/yourusername/KingsNet-IPTV-Pro-Web.git
   
2. Open the application
  
   cd KingsNet-IPTV-Pro-Web
   # Simply open index.html in your browser, or serve with a local server:
   python -m http.server 8000
   
3. Load your playlist
   路 Enter an M3U URL (e.g., https://iptv-org.github.io/iptv/index.category.m3u)
   路 Or upload a local M3U file
   路 Click any channel to start streaming!

 Supported Playlist Formats

路 Standard M3U format with EXTINF metadata
路 Supports tvg-id, tvg-name, tvg-logo, and group-title attributes
路 HLS streams (.m3u8) and direct video links
